# Web Map Trial

This repository contains a simple web map application built with React 18 and OpenLayers.

## Running the app

1. Install dependencies:
   ```bash
   cd web-map-app
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Run tests:
   ```bash
   npm test -- --watchAll=false
   ```
4. Build for production:
   ```bash
   npm run build
   ```
